# VideoLens 9 种分析模式 Prompt 模板

本文档为 9 种分析模式提供推荐 prompt 模板。用户可基于这些模板自定义 `--prompt` 参数。

## 1. general（通用分析）

```
分析这个视频的主题、核心观点、关键画面与时间线。
请输出：
- 视频主题（一句话）
- 核心观点（3-5 条）
- 关键画面时间戳与描述
- 总结（一段话）
```

## 2. bug（Bug 检测）

```
检测这个视频中展示的软件 Bug 与异常。
请输出：
- Bug 列表（每个 Bug 含：现象、复现步骤、可能原因、严重度）
- 异常行为时间戳
- 建议修复方案
```

## 3. ux（UX 分析）

```
分析这个视频中展示的用户体验问题。
请输出：
- 用户操作流程时间线
- 痛点与摩擦点（含时间戳）
- 满意度评估（高/中/低，含理由）
- 改进建议（按优先级排序）
```

## 4. tutorial（教程分析）

```
拆解这个教程视频的内容。
请输出：
- 教程主题与目标受众
- 步骤列表（每个步骤含：时间戳、操作、要点）
- 关键概念解释
- 常见误区与注意事项
```

## 5. product_demo（产品演示）

```
分析这个产品演示视频。
请输出：
- 产品名称与定位
- 核心功能演示时间线（含时间戳）
- 卖点提炼（3-5 条）
- 目标用户画像
```

## 6. meeting（会议总结）

```
总结这个会议视频。
请输出：
- 会议主题与时间
- 参会者（若可识别）
- 议题列表（每个议题含：讨论时间、关键观点、决议）
- 行动项（负责人、截止时间）
```

## 7. content（内容审核）

```
审核这个视频的内容合规性。
请输出：
- 内容分类（教育/娱乐/商业/...）
- 合规风险点（含时间戳）：涉政、涉黄、涉暴、侵权、虚假宣传等
- 风险等级（高/中/低）
- 处置建议（通过/警告/下架）
```

## 8. privacy（隐私检查）

```
检查这个视频中可能存在的隐私信息。
请输出：
- 隐私信息类型：人脸、身份证、手机号、地址、邮箱、IP 地址、屏幕敏感信息等
- 出现时间戳与位置
- 风险等级（高/中/低）
- 脱敏建议
```

## 9. production_recipe（生产配方）

```
提取这个视频中的生产流程配方。
请输出：
- 产品名称
- 原材料清单（含用量）
- 工艺步骤（每个步骤含：时间戳、操作、参数、注意事项）
- 质量控制点
- 成品规格
```

## 使用示例

```bash
# Bug 检测模式
videolens analyze /tmp/video.mp4 \
  --prompt "$(cat skills/videolens/assets/prompt-templates.md | sed -n '/## 2. bug/,/## 3./p' | sed '1d;$d')" \
  --mode bug \
  --output-dir /tmp/videolens_bug

# UX 分析模式
videolens analyze /tmp/video.mp4 \
  --prompt "分析这个视频中展示的用户体验问题..." \
  --mode ux \
  --output-dir /tmp/videolens_ux
```
